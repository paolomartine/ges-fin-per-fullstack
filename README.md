# 💰 Gestión Financiera Personal

![CI](https://github.com/paolomartine/ges-fin-per-fullstack/actions/workflows/ci.yml/badge.svg)

## 📌 Descripción
Aplicación web fullstack para la gestión de finanzas personales.  
Permite a los usuarios registrarse, iniciar sesión, administrar cuentas y registrar transacciones.

---

## 🛠 Tecnologías utilizadas

### Backend
- Java 21
- Spring Boot
- Maven

### Frontend
- React
- TypeScript
- Vite

### DevOps
- AWS EC2
- GitHub Actions (CI/CD)
